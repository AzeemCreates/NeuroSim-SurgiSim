#NeuroSim — Kean Hackathon 2026

Built a project in a team of three. This project idea originated from the fact that brain surgeons operate in a high-pressure environment with every surgery they have scheduled.

Building an interactive UI and game helps users build muscle memory and knowledge, which is why our application may be considered useful.

---

#How to Use NeuroSim

Users log in, explore the interactive 3D brain model, select highlighted brain regions, and receive real-time AI voice guidance from the Chief Neurosurgeon.

After completing a training module, users can connect a Phantom Wallet on Solana Devnet and claim an on-chain Certificate of Completion.
